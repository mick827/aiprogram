## 🔗 コース別申し込みリンク設定

各コースの申し込みボタンが個別に設定されています：

1. **基本コース申し込みボタン** (ID: `btn-basic-course`)
2. **伴走コース申し込みボタン** (ID: `btn-support-course`)  
3. **売上アップコース申し込みボタン** (ID: `btn-premium-course`)

**リンク設定方法:**
```javascript
// js/script.js の handleCourseApplication 関数内で
// 各コース別の申し込みURLを設定してください

function handleCourseApplication(courseType, courseName, price, installment) {
    let applicationUrl = '';
    
    switch(courseType) {
        case 'basic':
            applicationUrl = 'ここに基本コースの申し込みURL';
            break;
        case 'support':
            applicationUrl = 'ここに伴走コースの申し込みURL';
            break;
        case 'premium':
            applicationUrl = 'ここに売上アップコースの申し込みURL';
            break;
    }
    
    window.open(applicationUrl, '_blank');
}
```

**✅ 設定済みリンク:**
- 基本コース: https://mosh.jp/services/308181
- 伴走コース: https://mosh.jp/services/308186  
- 売上アップコース: https://mosh.jp/services/308187

申し込みセクションは簡潔なボタン形式に変更済みです。
